# handstand_pose

Minimalistic handstand pose checker using MediaPipe and OpenCV.

## Requirements

- Python 3.10+
- mediapipe
- opencv-python
- numpy
- pyttsx3

Install dependencies:

```
pip install mediapipe opencv-python numpy pyttsx3
```

## Usage

**Webcam mode:**

```
python main.py
```

**Video file mode:**

```
python main.py --filepath /path/to/video.mp4
```

Press `q` to quit.
